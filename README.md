# Setup
You need to have installed git, conan and premake

Clone the repository
```
git clone -r https://github.com/Prz-Ellam/Photoxel
cd Photoxel
```

Download premake and execute premake5.lua
https://premake.github.io/download

```
mkdir vendor
cd vendor
mkdir Premake
cd Premake
```
